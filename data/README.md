# Data Directory

This directory contains:
- CSV datasets for doctors, hospitals, and symptoms
- SQLite database files
- Trained ML models (pickle files)
- Sample data for testing

## Files Structure:
- `doctors.csv` - Doctor information and specializations
- `hospitals.csv` - Hospital locations and services  
- `symptoms_diseases.csv` - Training data for ML model
- `healthcare.db` - SQLite database (created automatically)
- `disease_model.pkl` - Trained ML model (created during training)
